## STEP 1: Update SDK Path (2 min)

1. Open Android Studio
2. Menu: File ‚Üí Settings (or Preferences on Mac)
3. Go to: Appearance & Behavior ‚Üí System Settings ‚Üí Android SDK
4. COPY the SDK path shown at the top

5. Open this file in text editor: /app/android/local.properties
6. Replace: REPLACE_WITH_YOUR_SDK_PATH
7. With your SDK path (examples below)
8. SAVE file

### Windows Example:
sdk.dir=C:\\Users\\YourName\\AppData\\Local\\Android\\Sdk

### Mac Example:
sdk.dir=/Users/YourName/Library/Android/sdk

### Linux Example:
sdk.dir=/home/YourName/Android/Sdk

**IMPORTANT:** Windows uses DOUBLE backslashes (\\)

---

## STEP 2: Build APK (15 min)

1. Open Android Studio
2. Click: Open (NOT "New Project")
3. Navigate to: /app/android folder
4. Click: OK
5. WAIT for Gradle Sync (5-10 min first time)
   - Progress bar at bottom
   - Don't close or cancel!

6. When sync complete:
   Menu ‚Üí Build ‚Üí Build Bundle(s) / APK(s) ‚Üí Build APK(s)

7. Wait 2-5 minutes

8. Notification appears: "APK(s) generated successfully"

9. Click "locate" to find your APK

**APK saved at:**
/app/android/app/build/outputs/apk/debug/app-debug.apk

---

## STEP 3: Install (5 min)

1. Copy app-debug.apk to your Android phone
2. On phone: Open Downloads or File Manager
3. Tap: app-debug.apk
4. If asked: Enable "Install from Unknown Sources"
5. Tap: Install
6. Tap: Open
